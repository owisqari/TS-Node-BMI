# TS-Node-BMI

## dependencies

- [Node.js LTS](https://nodejs.org)
- yarn

### to install yarn

`npm i -g yarn`

### to install typescript CLI

`sudo npm i -g typescript`


### to install node

`yarn add -D express ts-node`

## To install nodemon 

`yarn add -D nodemon @types/express typescript`

## To start application

`yarn dev`

## To run a demo browser
